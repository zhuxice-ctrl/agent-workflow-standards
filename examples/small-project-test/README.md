# Small Project ADworkflo Example

This example documents the minimum v2 lifecycle for a small application.

```text
PRD.md with FR-001
-> ARCH.md references FR-001
-> design_alignment.py analyze
-> independent semantic approval
-> configure task_spec.json
-> prepare_context.py
-> implement and verify
```

For layered development, generate `layer_plan.json`, customize the three four-question contracts, define cross-layer interfaces, and schedule capability slices through a validated execution plan.
