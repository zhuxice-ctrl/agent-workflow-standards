# ADworkflo Project Instructions

This project uses ADworkflo. For complex work, configure `.adworkflow/task_spec.json`, prepare bounded context, use run-scoped task artifacts for multi-task execution, and record verification before completion. PRD/ARCH work requires a passed `design_alignment_report`; explicitly requested layered development requires the three layer contracts in `layer_plan.json`. L2 work requires an accepted `context_preflight` before implementation and a passed post-edit `impact_report` before verification.
